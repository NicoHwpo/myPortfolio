3D VIDEOGAME

CORONALYPSE (2020) is a 3D first-person perspective survival videogame that I developed in 2020 for a school final project/competition. The main goal of CORONALYPSE is to collect all the vaccines in the map before you get too contaminated with the virus that the "zombie" like enemies have. You have to defend yourself with your shotgun. I used Unity with C# for the game-dev and blender for the 3D models. Play It!

PYTHON BOTS

This project is an Instagram comment bot that I made using python and the web-browser controller. This was a personal project which its main purpose was to comment in a giveaway post every x amount of time and post them. This bot is 100% automatic; it logs into an existing account, goes to the post, starts tagging 3 random people from an array, publishes the comment, and repeats the tagging and publishing steps.
Fun/sad fact: my bot commented over 2,000 times but I still didn't won the giveaway SMH.
